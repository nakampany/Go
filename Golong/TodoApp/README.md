## todoApp

## 使用技術
- Docker
- postgres
- go


## docker-compose コマンド
- build
  ```
  docker-compose build
  ```
- コンテナ起動
  ```
  docker-compose up -d
  ```
- app
  ```
  docker-compose exec app sh
  ```
- postgres
  ```
  docker-compose exec postgres bash
  ```

- docker down
  ```
  docker-compose down
  ```
